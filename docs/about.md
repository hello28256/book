# 关于本书

## 这本书是什么

《Docker 极简手册》是一本写给**已有几年开发经验、但还没系统学过容器**的后端工程师的小书。

- **不是**官方文档的中文翻译
- **不是**面面俱到的参考手册
- **是**一个老司机带你走 3 小时的实战速通

## 写作理念

- 📏 **极简**：5 章覆盖 80% 场景，剩下 20% 去查官方文档
- 🛠 **实战**：每一段代码都验证过能直接 copy 跑
- 🎯 **决断**：会明确告诉你"用 A 不要用 B"，不和稀泥
- 💬 **人话**：避免"容器编排引擎云原生 microservice"这种黑话堆砌
