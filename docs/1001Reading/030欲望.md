# 030欲望

**欲望，** 就是**内心对某种事物、状态或体验的渴求。** 简单说**想得到、想拥有、想体验，** 这就是欲望。

**欲望从哪里来，** 欲望源于生命本能。

- 人天生会追求食物、安全、亲密关系、尊重、成就。饿了想吃饭，累了想休息，喜欢一个人想靠近她，希望工作得到认可，想赚更多钱，这些都属于欲望。

**欲望的层次**

- 生理欲望，与生存和繁衍有关，如食欲、性欲、睡眠需求，这是最基础的欲望。
- 情感欲望，希望获得爱、陪伴、理解、归属感
- 社会欲望，希望得到地位、尊重、认可、财富，想升职加薪，希望别人认可自己的能力。
- 精神欲望，追求知识、真理、智慧、人生意义，阅读经典、研究技术、思考生命问题。

**欲望的作用**

- 欲望本身不是坏东西，如果没有欲望，不会学习、不会工作、不会恋爱、不会创造。
- 很多进步都来自于某种欲望。想解决问题，于是发明工具；想过得更好，于是努力工作；想理解世界，于是探索科学，**欲望是行动的动力之一。**

**欲望的问题。** 问题不在于有欲望。问题在于：

- 被欲望控制沉迷游戏、暴饮暴食、冲动消费、整天沉浸在幻想里，这时人不是在使用欲望，而是在被欲望牵着走。
- 欲望无限膨胀，得到一个目标后，立刻想要更多。于是长期处于焦虑、不满足、比较之中。

**如何与欲望相处，** 通常有三种方式：

- 放纵，想要什么就追什么，容易失控。
- 压抑，强迫自己什么都不要想，往往适得其反，越压抑越容易反弹。
- 观察和管理，承认欲望存在，然后问自己这是什么欲望？它合理吗？我应该立即满足，还是延迟满足？我想念女朋友。可以先区分是想发生亲密关系？还是想被陪伴？还是因为孤独？当看清欲望来源后，往往更容易处理。生存、责任、创造、挫折、欲望
