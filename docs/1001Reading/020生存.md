# 020生存

**生存，** 最基础的意思是：**活着，并持续活下去。**

**生物层面的生存。** 满足最基本的需求：吃饭、喝水、睡觉、呼吸、避免危险。这是所有动物都具备的本能，一个人在沙漠里寻找水源，是为了生存。

**社会层面的生存。** 人不仅要活着，还要在社会中立足。包括：工作赚钱、学习技、建立人际关系、获得资源。学习编程、找工作，本质上也是为了提高自己的生存能力。

**心理层面的生存。** 有些人身体活着，但内心很痛苦。因此心理上的生存还包括：希望、归属感、被理解、被尊重，长期孤独的人，虽然生理上活着，但心理生存状态可能很差。

**精神层面的生存。** 当基本生活解决后，人会思考：我为什么活着？我的人生价值是什么？我想成为什么样的人？这已经超越了“活下来”，开始追求“活得有意义”。有的人一生追求财富，有的人追求智慧，有的人追求帮助他人。

生存、生活、生命。可以把人生看成三个阶段：**先解决温饱，再提高生活质量，最后寻找人生意义。** 很多人年轻时忙于生存，中年经营生活，晚年思考生命。但这三个层次往往是同时存在的，只是侧重点不同。
