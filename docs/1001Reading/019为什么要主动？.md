# 019为什么要主动？

**很多事情之所以强调“要主动”，本质原因是现实世界不会自动向你靠近，资源和机会是分散的，主动是把“概率”变高的方式。**

![image-20260624110646980](https://oss-pai-y1p7zrgkn9rww3n1nb-cn-shanghai.oss-cn-shanghai.aliyuncs.com/image-20260624110646980.png)

**世界默认是“被动系统”。** 大多数事情的自然状态是信息不会自动到你面前、机会不会自动分配给你、别人不会优先考虑你。因为现实中存在三个事实。**资源有限。** 比如好工作岗位有限、好机会有限、好人脉有限，不主动等于默认被别人选走。**信息不对称。** 别人不知道你需要什么，你也不知道别人手里有什么，不主动等于永远停留在“信息盲区”。**注意力稀缺。** 所有人都很忙，老板不会盯着你成长、机会不会等你准备好、人不会主动理解你，世界不会为你“停下来”。

<img src="https://oss-pai-y1p7zrgkn9rww3n1nb-cn-shanghai.oss-cn-shanghai.aliyuncs.com/image%201.png" alt="image.png" />

**提高“接触概率”。** 可以理解成一个公式：成功概率 = 机会数量 × 被看见概率 × 准备程度，“主动”主要影响前两项，你接触更多机会，你被看见的概率更高。为什么被动的人容易吃亏？**被动的人通常是等通知、等机会、等别人来找，问题在于等只有极少数机会会刚好落到你身上。现实里大部分好结果来自投简历、主动沟通、主动表达需求、主动争取资源。主动不是“讨好别人”，而是“参与系统”。** 很多人误解主动，以为是低姿态、讨好、争取认可，其实更本质是你进入了游戏，而不是站在场外看，被动等世界给我安排，主动让世界知道我是谁、我要什么。
找工作：被动等招聘信息刷到合适的，主动投递 + 联系 + 复盘 + 内推，被动地找工作
人际关系：被动等别人来联系，主动沟通、维系关系。长期稳定的人际关系，一定需要双向参与，而不是单向维持。
学习成长：被动学习是老师教什么学什么，主动学习是自己找资料、问问题、实践，被动地学习很难成长。

<img src="https://oss-pai-y1p7zrgkn9rww3n1nb-cn-shanghai.oss-cn-shanghai.aliyuncs.com/image%202.png" alt="image.png" />

**主动其实是在对抗“熵增”。** 世界自然趋势是混乱、随机、分散。如果你不主动机会会散、关系会断、资源会流失。主动的本质是把分散的东西重新连接起来。因为世界不会主动为你优化路径，所以你必须主动去提高自己和机会的连接概率。
