# 003论人生短暂

很多人忙活了一辈子，还没享受人生，人生就开始过去了。人生其实不短，因为我们浪费了太多时间，不会利用时间，就好像你有很多的钱很多的财富，不会利用瞎用，那很快就会花光，你的人生，你的时间，如果你不会用的话一下子就会花光，如果你会用的话就不会觉得人生苦短了，而现实是人们常常为了没有头的欲望，或者是得到别人的肯定，甚至是别人的欲望，别人的财富别人的事情，在浪费自己的时间。设想一下别人抢占了你的土地，占了你的房子，抢了你的金钱，你一定会拿起自己的武器，夺回自己的东西，但为什么偏偏对于人生的时间这个财富，大家却毫不珍惜，随便浪费，甚至拱手让给他人，时间和金钱都是财产，那么我们为什么只保护金钱，不保护时间。

你要回想一下，自己的一天有多少时间是花在自己身上的，又有多少时间是花在别人身上的，很多人一天除了吃饭睡觉上厕所之外，剩下所有时间都浪费掉了，都花在别人身上，上班的人最为明显，早上一起来，就匆匆茫茫赶往车站，在路上就话不少时间，在公司忙一天到家倒头就睡，那么你这天除了睡觉之外，剩下的时间都是为别人所忙，有些人会说，我上班是为了赚钱，用我的时间去换了金钱，这不能完全为别人忙活。

那什么叫为了自己的时间，什么叫为了别人的时间，自己的时间就是自由，就是想干什么就干什么想做什么就做什么，做你想做的事情的时间，也就是不是做你自己想做，是被迫做的，身不由己做的，都不是为了自己的时间，而是浪费掉或者为了别人的时间。上班和上学也是一样，如果你喜欢的话这个时间就是为了你的时间，如果不是这样，是被迫去，那就是为了别人的时间，一天花在自己身上的时间并不多，睡觉并不能算为了自己的时间，他算是一种浪费掉的时间，只有做你自己想做的事情的时候，才是自己的时间，一天中有很多时间不是为了自己，很多人没有意识到这一点，是因为这些人不知道生命的终点其实离我们每一个人都不远，人在年轻的时候健康的时候，下意识认为自己这种年轻和健康的状态，会一直持续下去，大部分人都是年老，当疾病来袭，当灾难来袭的时候，才会意识到这一辈子的长度并不是你自己能决定的，你会意识到另一个问题，你的人生根本没有为自己活过，这一辈子都是为了别人瞎忙活，当意识到这个问题的时候，往往就已经来不及了，没有人的人生可以重来，不要对未来抱有任何期望，因为死亡和明天不一定哪一个先来到，死亡不是在你完全不经意的情况下，它就突然到来了，所以你说我年轻的时候多努力努力，等我三十岁了四十岁了干这干那，谁告说你，你能活到三十岁四十岁，活到你有钱的时候呢，而我们真正能控制的事情只有一个就是今天，或者此时此刻，所以我们应该珍惜生命中的每一天，每一天都要为自己而活，而且要把每天一天当成生命中的最后一天来活。就算你能活到明天，或者能活得很长，还有一个问题，就是时间的推移不是由我们控制，不论这一天你是否为自己而活，这一天都会过去，你的时间会不断向人生的终点推进，不管你的人生有多远，推进的速度不会改变，一刻都不会停歇，所以你要趁早把每一天都花在自己身上，自由去支配你的时间，做你想做的事情，不然一定会后悔，人生真正的长度是什么，就是你花在自己身上的时间的长度，人生的长度其实跟寿命的长短并没关系，有些人年纪轻轻三十多就离开人世，但是如果他人生中每一分每一秒都花在自己身上，他的人生就是三十年，而有些人活八十岁，一辈子都在为别人打工，为别人操心，那他的人生可能不到二十年。

一定要趁年轻的时候尽可能把更多的时间花在自己身上，做自己想做的事情，因为年轻的时候的一年和年老时候的一年，价值是完全不同的，很多事情年老了就不再有机会去做了。时间这个财富分为三个部分，叫过去、现在、未来，过去不可改变，未来不可预测，不可期待，而现在虽然很重要，但是太过短暂，它也在不断变成过去，所以人生真正唯一积攒下来的财富，只有一个东西，就是过去，你可以享受现在，但是人生积累下的财富其实是过去，也就是你的记忆，所以不能留下记忆的事情不要做。喜欢打游戏，每天打游戏，打了三十年，回忆这三十年，一句话就可以带过了，人生单调也是人生短暂的一个主要原因，人这一辈子应该体验各种各样的事情，思考各种各样的问题，来充实你的记忆，充实记忆就是充实人生。什么都体验过的，都经历过的人不会畏惧死亡的，哪怕死亡有一天突然到来，他也会欣然接受。如果你不把时间花在自己身上的话你就不会有属于自己的故事和记忆，你就没有自己的人生，你的人生就会很短。
