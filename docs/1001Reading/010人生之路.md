# 010人生之路

人生不是没有选择是选择太多了，必须要设定一个标准，在标准中选。没有人生之路，往往不是没有路可走，而是路太多。有一个误区就是人生之路必须成为一种工作，能够赚钱才是人生之路，不能赚钱就不能成为人生之路，你想不想做的是事情和成为工作其实是两码事，你最需要考虑的事情是你想不想做，能不能成为一个工作，能不能赚钱，这是社会问题。

为了帮助你选择的话，我们如何设定这个条件，就非常重要了，在很多事情上没有原则，如何设立原则，要设定原则，只需要考虑一个事情，就是你要了解你自己，你要了解你自己，而这个就是最难的事情，我们完全不了解自己，我们看别人可能还清楚一点，但对我们自己完全不了解。人们下意识会把愿望理想，作为人生选择最终条件，大部分是无法实现的，按指标重要性排序的话，第一个是价值观，第二个是才能，第三个才是愿望和梦想。价值观是做一个事情，最看重的东西 ，也就是什么事情不做，什么事情做叫做价值观；只要这个事情有钱我就做，这是一种价值观，如果这个事情对自己有益，我就可以做，这就是一种价值观，每一个人的价值观都是不一样的，但它对我们的选择起到了致关重要的作用。
