# 029主动改变还是被动改变？

很多人与人的差距，未必在智商、学历、出身，而在于：**面对变化时，是主动改变，还是被动改变。**

**主动改变的人特点是在问题还没逼到眼前时，就开始调整自己。** 身体还没出问题，就开始运动。工作还稳定时，就开始学习新技能。关系刚出现苗头，就开始沟通。发现自己焦虑，就主动调整生活方式。这种人不一定聪明，但通常拥有一种意识——我要对自己的人生负责。所以他们的人生轨迹往往比较平滑。

**被动改变的人特点是不到最后一刻不动。** 医生说血压高了才开始运动。被裁员了才开始学习。快分手了才想沟通。身体垮了才开始养生。这种改变不是不改变。而是被现实逼着改变。代价通常更大。

为什么很多人选择被动改变？因为改变本身有成本。**主动改变意味着付出时间、付出精力、面对不确定性，而不改变的好处是轻松、熟悉、安全，所以人天然倾向于维持现状。** 有些人活在惯性里，有些人活在觉察里。活在惯性里的人重复昨天、很少反思、被环境塑造，活在觉察里的人经常反思、愿意改变、主动选择方向。心理学里有个概念叫：现状偏见，意思就是人更愿意维持现状，即使现状并不理想。

**但现实有一个规律，你不主动改变，生活也会逼你改变。** 不锻炼身体，逼你改变；不学习，行业逼你改变；不经营关系，关系逼你改变；不面对问题，问题逼你改变；**所以人生并不是改变和不改变，而是主动改变和被动改变，也就是说没有不变这回事，无论是先改变和被动改变都是要改变。**

**健康生活、成长、工作与生活平衡、人际关系经营，这些本质上都是主动改变的课题。** 长期工作压力大、焦虑、睡眠不好、关系紧张，那么迟早都会面临调整。区别只是主动改变，现在开始调整工作节奏，学习放松、经营关系、照顾身体，还是被动改变，等到身体出问题、情绪崩溃、关系破裂再被迫调整。

但还有一个更深层的问题，**主动改变的前提是什么？不是能力。不是学历。而是承认问题存在。很多人改变不了，不是因为不知道怎么做。而是不愿面对问题、不愿承认问题、觉得自己没问题、所以成长的第一步往往不是行动，而是觉察。**

如果用一句话总结：**人生最大的差别之一，不是会不会改变，而是在问题还小的时候主动改变，还是等问题变大后被现实推着改变。主动改变的人，通常是在塑造人生；被动改变的人，更多是在应对人生。两者最终都会经历变化，但付出的代价和获得的主动权，往往完全不同。**
