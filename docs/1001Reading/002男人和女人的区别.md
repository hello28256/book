# 002男人和女人的区别

男人说话的时候主要用左脑，女人左右都用。左脑主要用于理性工作，如语言、文字、逻辑推理、判断；右脑主要负责一些感性的工作，如情感、图像、直觉、音乐。

男性说话的时候比较注重理性，注重逻辑性的，都是为了一个统一的结论服务的 ，很少会延伸到其他问题上；而女性说话的时候是左右脑共同工作的 ，说话的时候没有固定论点 ，会延伸到很多的话题上面，这些话题可能完全没有关系，也就是思维跳跃 ，说这说这就跳到一个完全没有关系的话题上去；所以女性聊天总有聊不完的话题，而男性聊天容易集中在一个话题上，如果大家对这个一个话题没有兴趣就聊不下去。

如果不能理解男性和女性在说话上这个区别的话 ，就会产生很严重的困惑，男人就会觉得女人说话的时候怎么老是岔开话题，而女性会觉得男性怎么总在一个话题上反复的说，好像在不停的讲道理，难道我不懂道理吗。对于男人交流是一种手段，他的目的是为了把自己的思想传递给对方说服对方，而女性聊天就是为了说话，结论不重要，过程的愉快很重要 ，他们希望在聊天的过程中产生情感的共鸣，这是男性不追求的。

男性是为了结论上的统一，女性是为了情感上的统一，产生共情。

男生尝试和女生讲明白一个道理这个想法就是错的，首先的要做的事情，就是让这个女生理解你到情感，理解你的处境，而不是理解你的理论。这个女生如果能体会你的痛苦，你的处境，你的心酸，她就会理解你，女生不是不懂道理，而是太懂道理，不要反复重复了。
