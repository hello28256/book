# 015《心经》

观自在菩萨，行深般若波罗蜜多时，照见五蕴皆空，度一切苦厄。舍利子，色不异空，空不异色。色即是空，空即是色。受、想、行、识，亦复如是。舍利子，是诸法空相，不生不灭，不垢不净，不增不减。是故空中无色，无受、想、行、识；无眼、耳、鼻、舌、身、意；无色、声、香、味、触、法；无眼界，乃至无意识界；无无明，亦无无明尽，乃至无老死，亦无老死尽；无苦、集、灭、道；无智亦无得。以无所得故，菩提萨埵，依般若波罗蜜多故，心无挂碍。无挂碍故，无有恐怖，远离颠倒梦想，究竟涅槃。三世诸佛，依般若波罗蜜多故，得阿耨多罗三藐三菩提。故知般若波罗蜜多，是大神咒，是大明咒，是无上咒，是无等等咒，能除一切苦，真实不虚。故说般若波罗蜜多咒，即说咒曰：揭谛揭谛，波罗揭谛，波罗僧揭谛，菩提萨婆诃。

色：在佛学指的是一切有形有相的东西，物质、身体、形象、外在的一起、你的房子、你的车子、你的脸、你的职位、你的名片，这都是色 。色即是空，空即是色。石头就是石头，桌子就是桌子，你踢到它脚会疼，这是事实，心经否定的不是物质本身，而是赋予物质的那个"意义"，他不是物质本身，它是你的大脑制造出来的，它会变、会碎、会过期。人们执着的不是物质本身，人们执着的是所物质承载的那个"身份"和"意义"。人们把一堆零散的东西，工作、收入、社交圈、外貌拼在一起，然后自动脑补成一个完整的"我"。这个"我"其实是大脑制造出的幻象。一个人早上照镜子，突然发现眼角多了一条皱纹，他的心情瞬间坏了，但那条皱纹昨天就在那里，他昨天没有注意到，他执着的从来不是身体本身，他执着的是头脑中那个"身体应该是什么样子"的影像，当现实和那个影像对不上的时候，他的痛苦就来了。心理学有一个概念叫"享乐适应"，研究表明买一辆新车带来的快乐只能持续大约72小时，三天之后，人的快乐水平回到了买车之前的基准线，然后他开始想要下一辆更好的车，这就是"色蕴之执"的陷阱，人们以为抓住了外在的物质就能留住快乐，但快乐从来都是留不住的。不是通过"拥有"来获得意义，开始通过"创造"来和物质世界打交道，从我"我应该拥有什么"变成了"我可以创造什么"，不是放弃了物质，而是改变了跟物质的关系。

受：受想行识，亦复如是。"受"在佛学里是指人的感受，佛教把感受分成三种：乐受（舒服的感觉）、苦受（不舒服的感觉）、舍受（不好不坏的中性感觉），人类天生趋向乐受、逃避苦受，这是写在基因里的程序，没有什么问题，问题出在第三种"舍受"，舍受就是那种什么都没发生，什么特别的感觉都没有的状态，绝大多数人完全无法忍受这种状态。生物学编程告诉人们，人应该有反应、有刺激、有情绪波动，就说明人在"浪费时间"或者"活着没意义"，《心经》指出的要害就在这里，人痛苦的最大原因不是"感受到了坏的东西"，而是"无法忍受感受不到好的东西"，很多人不是在追求快乐，而是在逃避平淡，而平淡恰恰是生命的底色，人99%的时间都处于舍受状态，不好不坏、不悲不喜，但人们把这99%的时间都当成了"需要被填满的"空白，所以人们不停地刷手机、不停地找刺激、不停地制造情绪波动，这就是"受蕴之执"，它源于对平淡的恐惧。你的大脑不允许你总是快乐，多巴胺系统的设计原理就不支持持续的快乐，但是你可以在不快乐的时候，依然好好地活着，你可以在悲伤来临的时候，让它来。你有没有这样的夜晚，工作不顺、天气不好、跟伴侣有点摩擦，你觉得心理堵得慌，于是你做了一个你非常熟悉的动作，打开手机开始刷短视频，一刷就是三个小时，三个小时后感到的是一种比之前更深的空虚，因为你不是在找快乐，你是在逃避，逃避那种平淡，人们逃避的是那个没有刺激的、安静的、空白的自己，允许自己不开心，可能是通往平静的最快路径，当你不再跟不开心较劲的时候，你才有精力去注意到生活中那些真正存在的微小的、被你忽略的东西，悲伤来了就让它来了，它会走的，你不赶它，它走得更快。

想：想蕴之执，"想"在佛学中指的是"概念化"，你的大脑在给一切东西贴标签、分类、下定义。比如好人，坏人、对的，错的，"无眼耳鼻色身意，无色声香味触法"，他不是说物理上没有眼睛，而是说你的眼睛不是在"接受"信息，而是在"分类"信息，你看见一个人，你的大脑在零点几秒内就完成了一系列分类操作："男性"、"越40岁"、"穿着不太讲究"、"可能是蓝领"。一旦你给某个东西贴了标签，你就再也看不见"它本身"了，你只看得到那个标签，这就是想蕴之执的陷阱。人的大脑有一套预设的分类系统，它决定了你能"看到"什么、不能"看到"什么，你以为你在客观地观察世界，你是透过一面印满标签的玻璃在看世界，你看到的一切都是被你标签系统筛选过的，心理学中有一个概念"确认偏误"，人类大脑天生只接受符合已有信念的信息，"无眼耳鼻色身意，无色声香味触法"，这里的法不只是佛法，也指一切用来组织世界的概念和规则，你的立场、你的价值观、你的审美标准，这些全是"想蕴"的产物，他们是工具，不是真理，当你把工具当成真理的时候，工具就变成了牢笼，你因该有立场，但你不必跟你的立场融为一体，"我持有这个观点"和"我就是这个观点"是两回事，前者是自由，后者是囚笼，你可以随时更新一个工具，但你不可能拆掉一面你以为是自己身体的墙。

行：行蕴之执，在现代社会几乎被视为最高美德。在佛学里，"行"指的是心的造作，你内心自动生成的冲动、反应模式、意志倾向，你还有决定做什么之前，你的心就开始造作了，已经在推你去做点什么了，不是你决定要行动，而是一股冲动推着你去行动，大多数时候是这股冲动在控制你，《心经》直接说"无苦集灭道"，因为"我要修行"，"我要解脱"，"我要变得更好"，"我要开悟"，这些听起来无比高尚的冲动，本质上跟"我要赚更多的钱"，"我要升职"，"我要变瘦"是同一种心理结构，本质是一样的，现在的我不够好，我必须做点什么来改变，"我不够好"这句话是行蕴的发动机，你所有的忙碌、所有的努力、所有的"上进心"，你以为他们的驱动力是追求卓越，很多时候，他们的驱动力是"我不够好"的恐惧，一个拼命加班的人和一个拼命打坐的人，如果他们内心的驱动力是"我现在不够好"，那他们戴的是同一条锁链，只是挂的招牌不一样，一个写的是成功，一个是开悟。人类所有的不快乐，都源于一件事：不知道如何安静地待在房间里，大多数人静静地坐三分钟就开始浑身不自在，心里会冒出一个声音："我是不是在浪费时间？我是不是要去做点什么，我还有好多事没做呢"，那个声音就是行蕴，它24小时不间断运行，它的核心指令只有一条："做点什么呢，什么都行，但不能什么都不做"。存在主义哲学家海德格尔有一个概念叫"被抛状态"，人是被"抛"进这个世界的，没有人问过你愿不愿意来，你来了，你就被要求"有所作为"，整个社会结构、教育体系、评价标准都在告诉你同一件事："你必须要做点什么来证明你的存在是有意义的"，如果这种"必须有所作为"的冲动本身是一条锁链呢？《心经》不是说你不应该做事，它是在问你："你做事的驱动力是什么"，你是看清楚了之后才选择做的，还是被一股你自己都说不清楚的动力推着做的？一个是自由，一个是上瘾，取决不在于你做不做，而在于你停不停得下来。

识：无眼界，乃至无意识界。连意识本身，连那个知道者，连那个你以为是最后堡垒的我，都是空的。大脑中有一个叫"默认模式网络"的系统，当你什么都没有想的时候，比如发呆、走神、无聊的时候，这个系统反而最活跃，它在想你自己，"我是谁"、"别人怎么看我"、"我昨天说的那句话对不对"、"我明天要干嘛"、"我这辈子到底要什么"，这些念头在你没有思考任何事情的时候，会自动地、不间断地涌现。禅宗六祖惠能问过一个著名的问题："你未出生之前的本来面目是什么？"，这个问题是让你看见，你从来就不需要一个"面目"，你本来就是完整的，不需要任何身份、标签、定义来让你"成为"你自己。无无明，亦无无明尽。没有不知到自己是谁的迷茫，也没有终于知道自己是谁的开悟，两者都是识蕴在运作，我不知道我是谁是一种执念，我终于找到了真正的自己也是一种执念，它们是同一条锁链的两端，真正的自由不是找到了自我，真正的自我不再需要特定的标签定义"自我"。
