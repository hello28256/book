# 007天天快乐

我们总是感觉到不快乐，倒不是真正因为我们遇到了什么困难或者什么不开心的事情，而是我们的大脑被设计为很难感受到快乐。大自然的动物表现出来的情感多为警惕、愤怒、恐惧、威赫，基本上没有快乐的情感。因为快乐对于生存来说并不重要，快乐可以让你生活更好，但是不能让你活下来，而什么东西可以让你活下来，是恐惧、警惕、威胁、恐赫，这些恰恰都是负面情感，这就是动物不会笑的原因，因为自然界不需要笑这个情感。

## 【不断的感知危险和预测危险】

我们作为一个动物，我们大脑有一个非常重要的工作就是不断的感知危险和预测危险，而这种不快乐和客观条件没有关系，和你有多少钱，和你长的帅不帅，和你学历高不高，一毛钱关系都没有，这是所有人类都会有个一个问题。而且目前已经基本确定一个非常可怕的事实，就是大脑的不快乐的状态有可能是我们大脑的默认的工作状态。人在清醒的时候有47%的时间是在走神的，就是干着什么事情，但是脑子在想别的事情，或者在发呆，这个时间和起来大概占清醒时间的一半，这种状态被叫做身心漫游。人在这种漫游状态下就是会感觉不到快乐的，当你没有主动思考一个事情的时候，就是精神涣散的时候，大脑就会进入默认的工作状态，就开始预测危险，想象各种可怕的事情发生，这我们的一种本能。所以只要一走神就很容易不快乐，而我们越忙呢就会越快乐，事实证明休息的时候可能比忙的时候更不快乐，如果你休息的时候是去其他喜欢的事情，进入另一种状态那就没有问题，问题是一旦没有集中，一旦走神了，大脑是绝对不会浪费这个时间，它会拼命的预测各种危险。

## 【负面偏见或者负面偏误】

为了快乐，我们减少这种闲暇时光是远远不够的，因为我们大脑还有一个问题，这个在心理学上叫负面偏见或者负面偏误，人往往会注意到负面的信息、负面的新闻，更容易关注坏的事情，更容易对他们产生强烈的情感，也更容易对这些不好的事情心心念念，让他们看起来比实际的影响更大，这里就是为什么热搜榜前面大部分都是不好的消息的原因，哪怕现实中80%都是好消息，20%是不好的消息，这20%都能上热搜，我们做事情的时候大部分时间都是顺利的，只要有一个小小的失误小小的不开心，我们也会关注在这个地方，无限放大，而忽略掉所有顺利的部分，这是我们的本能造成的，就是我们对于危险和负面信息更为敏感一些，在远古时期这些负面的信息对于我们生存来说更为重要，只是到了现代社会，我们可能不太需要关系那些负面的信息，但是我们这种思维模式已经更深蒂固了，很难改变，我们作为一个生物从客观和主观上来说都是很难快乐的，我们对于外界的危险和坏的消息更为敏感，这是客观因素，就算没有这些客观因素，我们闲下来也会开始想象一些危险的事情，所以客观和主观来说我们都是很难快乐的。要想改善这种给我带来不快乐的情况，想要改变思维模式有点困难，但是我们可以通过一些其他的手段，比如说尽量少看一些负面的信息，比如说减少之间的闲暇时光，让自己忙起来，干点自己想干的事情，与给自己带来负面情绪的人拒绝来往

## 【自我肯定感丧失】

还有一种情况会造成我们不开心，而这种不快乐更为持久更为严重一些——自我肯定感丧失，简单来说自我肯定就是满足于现在的自我，认为自己存在是有价值和意义的。没有自我肯定感最直接的表现就是一种严重的自卑，无法接受接受自己的存在，看谁都比自己好，人无法接受自己，喜欢自己的话就会生活在痛苦之中，最终产生抑郁。丧失自我肯定感还有一个表现就是无法感受到快乐，就是别人表扬你肯定你，你会觉得这是恭维，不是事实，丧失自我肯定感还有一个明显的表现，就是什么都不想做什么都不敢做，他们认为他们做什么都会失败，时间长了就不能上班上学了，最终就恶性循环，对生活放弃，对自己放弃。自我丧失跟你的真实环境没有任何关系，哪怕再有钱也会陷入这种自我丧失。自我肯定感可能会随时间推移改变，由下列原因直接引发自我肯定感丧失。

1. 第一个就是失败，人生很难一帆风顺，总是会遇到失败，当遇到失败和挫折的时候，尤其是不断出现的失败和挫折的时候，人就会不断丧失这种自我肯定感；导致自我肯定感丧失的
2. 第二个原因外界的否定，当人一直生活在否定的环境中的话，就会开始丧失自信；
3. 第三个原因比较，哪怕你没有什么悲惨的过去，也没怎么失败过，但是你和周围人一比，你就会不断丧失自我肯定，大部分人都想和好的人比，不愿意与差的人比，很容易造成心态奔溃，只能说大家不一样，千万不要比。

这三个问题都很难直接解决掉，因为我们无法忘记过去，因为我们很难堵住别人的嘴，让他们不否定我们，因为我们很难不去比较，就算你再优秀，你总有比不过的地方，你再完美，终究是有缺陷，这个缺陷会被无限的放大。这个人类社会的不完美，造成我们没有办法彻底规避这是三个问题，过去就在你大脑里，有些人说不想过去，实际一直在想过去，你不能不断的回想你的失败历史，他会不断的消磨你的自我肯定感，消磨你的自信。不能直接解决，其实是有一些间接的方法，

1. 尝试表扬自己，站在镜子前表扬自己，比如你真帅，这种表扬是没有根据的，一旦有了根据就产生了比较，没有根据的自信是完美的，你就认为自己帅是完全没有问题的，越有根据的自信就越不成立，就越容易被击溃，因为你永远不可能你是最厉害的那一个，你觉得自己长的帅，自己长的高，自己有钱，这是一种自我认知，你觉得自己是就可以了，有些女孩子的自信来源是自己年轻漂亮，但人终究有不年轻的那一天，如果这个为根据，那一旦那一天到来的时候，你马上回变得非常悲观，自我肯定感会马上消失，你的自信会荡然无存，所以的自信最好来自于没有根据，就是自己觉得就行。自信来自没有根据的地方是最有用的，自己认为自己怎么样这是有实际用处的，当然别人能给你这种自信，这是最有效的，但是不能指望总会给你这种这种正面的反馈，别人没有义务给你这种正面的反馈，所以你必须要给自己这种正面的反馈，这就是为什么要不断提升自我肯定感的目的，并不为了是获得成功，而是让自己感觉轻松和快乐，只要自己觉得幸福那就可以了。
2. 不要使用负面词汇，使用正面词汇，不要说："我不想失败"，而说："我们想要成功"，不要说："糟了糟了，完了"，而是说："好的"，不要说："怎么办怎么办"，应该说："应该是有解决办法的"，塞翁失马焉知非福，这个世界是这样运作的，不是你认为好的结果，他就是好的，你认为不要的结果就是不好的，所以完全不要因为一个你认为的负面结果不断压垮自己，我们面对问题要表现积极态度，是因为消极也是没有用的，其次这个世界大部分事情是不能按照计划进行的，这是一种常态，面对挫折面对困难时一种常态，反倒一番风顺时不正常的，对于一个正常的事情和常态的事情，你会说糟了糟了完了完了吗，你会说："好好，我知道了"，这里只是用一种正常的心态，去接受这个世界上正常的事物。并不是自欺欺人，去欺骗自己，面对失败的时候说我欺骗自己，完全不是欺骗，而是一种正常表现，因为失败是正常的。
3. 还有一个间接解决自我肯定感丧失的方法是尝试放下过去，放下过去很困难，但还是要放下，尝试原谅自己，尝试原谅别人，不能原谅自己，不能原谅别人的话，你就会永远困在过去，你就会总是想起这个事情，一想起就会很生气，如果不想生气那就原谅，如果不能原谅那就忘记，如果被困在了过去，你就没有现在，就没有明天，除了忘记过去，还有一个重要的事情，就是告诉自己在这个世界上除了活着没有更重要的事，无论你现在觉得多么重要的是，过几年回过头来看都没有那么重要，所以没有必要纠结，这个事情很重要，我一定要去成功，我一定要去完成它，没有必要。如果想要彻底解决这个问题，必须要养成一种正面思维的习惯，为什么我不能那么想呢，其实不是，而是还没有形成习惯而已，时间未到。
4. 用喜欢的力量，如果每天都做同样的事情，吃同样的东西的话自我肯定感就会慢慢下降，需要增加新鲜感，没有新鲜感的人生就会负能量，自我肯定感越来越低。
5. 尝试把你的想法写下来，把问题写出来，就从心里看转变为上帝视角来看待，立马提高一个纬度。
6. 学会把自己从困难中抽离出来，当遇到这个困难的时候别人是如何解决的
7. 真正的自我肯定，不管平时做了一点好事情，正确的事情，顺利的事情，都要给于自己正面的反馈和表扬，告诉自己你真的做到了，做得很好啊。很多人把标准设定太高了，改善最好的方法就是把自己设定为标准，自己是普通的那一个是周围的人太优秀，生活在一堆优秀的人之间，多么幸福的事情啊，何必把标准定太高，结果一帮笨蛋。

人要活在当下，为什么要活在当下，因为只有活在当下的人最幸福。

缓解压力最好的方式是读书，听音乐，喝红茶、散步、打游戏。
