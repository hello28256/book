# 012《被讨厌的勇气》-阿德勒

人生如何获得幸福，阿德勒心理学三个核心理论：

## 目的论

这个理论和"心理创伤"有关，以前经历过一些挫折，痛苦，或者生活巨变，造成你产生心理障碍，一朝被蛇咬，十年怕井绳。人的现在是由现在的目的所决定的，这个目的有可能存在你的潜意识当中，就是说你有可能没有意识到你有这样的目的，但是你的大脑已经开始为你的目的开始寻找理由了，它上哪里去找理由呢，就是上你的记忆中去找 ，找到一个理由，然后把他合理话，让你觉得是这个原因造成了这个结果，而这个原因根本不存在过去，而是你现在的目的，想要治疗你的心理创伤是改变你现在的目的 ，而不是寻找过去的源头 。举个例子，就比如你想去告白，但是最后你没有去告白，你说是因为你一紧张就结巴所以不能告白，按照弗洛伊德的理论，要想让你告白，必须解决你一紧张就结巴这个我哪天 ，解决了这个问题你就可以告白了，但是阿德勒认为，你不想去告白是因为你压根就不想去告白，你要想告白跟你结不结巴没有关系，你一定回去告白，为什么不去告白，是因为你不去告白更舒适，符合你现在的生理需求，所以你不去告白，你接受这种现状，所以你找这种理由，说我结巴，就算你不结巴，你会找其他理由。弗洛伊德的的因果论是相当悲观的，因为你的现在是由过去决定的，而你的过去又很难改变，所以你只有后悔的份，相反阿德勒的目的论就相当乐观了，他认为你的现在是由你现在所决定的，所以你要改变你的人生只需要改变你现在的目的就可以了，也就是说阿德勒的理论是抚平伤痛，而阿德勒的理论是建立勇气，改变现状的勇气，他不仅认为心理创伤不存在，他认为恐惧也好，自卑也好，都是我们逃避现实的一个工具。而且人的现在可以反过来影响过去，爱因斯坦小时候很傻的，上学的时候连小板凳都坐不好 ，所以被同学嘲笑，被老师嘲笑，那么后来爱因斯坦成为这个世界上最聪明的人，他坐不好小板凳这个事情就成为了一段佳话，同样坐不好小板凳，如果爱因斯坦后来真的是一个傻子了，那么这个就是心理创伤。人们对同一段历史的解读会因为后来的结果不一样而不一样，所以过去是有可能被改变的，总结成一句话就是阿德勒认为人是可以改变的，而且很容易改变的，只是不想去改变。

## 课题分离

这个是处理人际关系的一个非常核心的理论，阿德勒认为人的所有烦劳都来自于人际关系。你觉得自己穷，是因为你见过富的，你觉得自己学习不好是因为你班上有学习好的，你觉得自己长得不好看是因为有长得好看的，所以你烦恼的根源就是和别人比较，这也就是为什么人际关系是所有麻烦的源头。怎么解决人际关系这个问题呢，就要用到阿德勒这个课题分离，当你遇到一个问题的时候，你要把这个问题分为两个问题，一个是你自己的问题，另一是别人的问题，而你只需要解决自己的问题，不能够干涉他人。就比如你要辞职去做Youtuber，你老婆不同意，这个事情是两个问题，一个就是你要辞职，另一个是你老婆不同意，按照课题分离的理论，你只需要解决你自己的问题，就是你只要去辞职，你老婆同不同意是她的问题，你不能介入，反过来从你老婆的观点，她只要解决一个问题就是，如果你辞职的话，她继续和你过，还是改嫁他人，他不能够干涉你要不要辞职这个决定，如果你们开始顾及对方的感受，那么这个问题就解决不了，你要辞职，你老婆要真改嫁了，那是她的自由，你要承担这个后果，你要不想承担这个后果，你就不要辞职。阿德勒认为人际关系产生问题最大的原因有两个，一个就是你介入他人的问题，另一个就是他人介入你的问题，所以要解决人际关系的问题，就要把它两彻底分开，人为什么会介入他人的问题，就是为了的到一种自我心理满足，通过影响他人来实现一种控制欲，这老公辞职这个例子中，老婆之所以不同意老公辞职的原因，如果老公辞职生活就会变得不稳定，生活变得不稳定这是老婆的事情，老婆唯一要做的事情就是判断要不要和他继续在一起过，你要和他过就去找一份工作，贴补家用就完了，或者你选择改嫁，这都是你的自由，你要做的是要不要辞职，如果最终没能够辞职，你这一辈子都会记得这个事情，以后可能成为你的一个爆发点，如果你老婆屈服于你，去另找一份工作，和你过下去，最后没有成功的话，这也会成为她的一个结。所以永远都不要强调我为了你都这样这样这样了，也不要强调你能不能为我怎么怎么怎样，千万不要去最求这个，一旦追求这个问题，人际关系就会变得很麻烦，总之你自己的事情只能你自己做，你也要承担相应的后果。阿德勒不是希望每一个自私，而是希望每一个人他自己的内心变得强大起来，不依赖于其他人，因为我们不能指望他人对我们负责，我也不能替别人找想，有些人说我这样是为了你，但是其实我们并不能够为他人设身处地的着想，因为你不是他。

阿德勒强调父母不要表扬孩子，老师也不要表扬学生，表扬就是一种干涉，表扬是一种洗脑，而且是二级洗脑，洗的是潜意识，它会在孩子内心深处形成一个，我学习我努力的目的是为了获得表扬的这么一个种子，为了这个而学习，而不是真正对这个东西感兴趣，而这种洗脑会造成他们认为自己存在的价值，是别人的肯定，而不是自己本身，所以学校会产生强烈竞争的原因就是在这，如果我们不是为了追求别人的认可而存在的话我们应该如何存在呢，这就是阿德勒的第三大理论。

## 共同体感觉

就是我们希望成为一个群体当中的一个重要的部分，阿德勒认为人不应该追求得到他人的认可，而是追求共同体感觉，也就是说我们不应该追求的到某一个人的认可，而是应该追求在一个集体当中担当一个重要的职位 ，这个其实就是我们如何感知自我存在的价值，我们存在的价值并不是通过别人的认可而获得的，而是通过对集体的贡献而获得的，所以阿德勒认为任何人不应该追求竞争，他说竞争是一种没有自信的表现，你要通过和别人的比较才能获得自我的价值，也就是通过别人的认可，才能得到自己的价值的话，那你存在就没有意义，所以真正的高手从来不与别人相争。他人认可和共同体感觉最大的区别最大的区别是，如果你是为了他人的认可而存在的话，那他人不存在的时候，你就不会行动，就比如做好事，周围没有人看我就不做好事，因为我是得到别人认可我才做的，如果是为了集体的利益，那么他就不会在乎别人有么有看，如果你是为了获得别人的评价在努力的话，那么你就永远无法获得自由，你被他所束缚，共同体感觉就相反了，为了自己而活，我认为这是对集体做贡献我就这么做了，别人评价根本不重要。夫妻之间也属于一种团体共同体，如果这个共同体都是为了对方而活着，会非常的累，如果夫妻目标相同的话还好，但是目标不同的话会非常累，说我为你牺牲这么多，你又反过来说你又为我牺牲那么多，两个人的争论是永远都不会停止的，但是两个人都能够为整个集体着想，做出正确的行动的话，那么两个人之间就不会有任何矛盾的同时集体也能正常发展，所以真正的夫妻之道是应该，两个人都为了两个人好而努力，而不是为了对方而努力。所以之前那个例子老公要辞职，老婆应该立刻去找一个工作贴补家用，就是两个人做出所有决定都是为了这个共同体而在努力，怎么才能有这种共同体感觉呢？阿德勒提到三个步骤：

1. 接受自己，你要完全接受自己的存在，有很多人认为当别人不在评价的时候，他就觉得自己没有价值，换句话说就是任何人都不存在了，你都是认为自己是存在的才行，这是潜意识的一部分，很多人看上去觉得自己存在，但是你仔细想想你是不是为别人而存在，就是有很多孩子是为了父母而存在，根本就不知道自己为什么活在这个世界上，有很多上班人也是，我就是问了公司而存在的，或者就是为了家里的老母亲，或者老婆而存在的，这就完了，你不能接受自己的存在的话，你就无法体会到共同体感觉，当你完全接受自己的存在，你就完全不会理会他人的目光，他人的评价就不重要了，因为他们在不在，你都会在。
2. 信赖他人，这个信赖是无条件的，你不走出这一步，别人也不会信任你，你必须无条件信任他人，他人才会信赖你，这样才会组成一个共同体。
3. 为共同体作出贡献，你不能干涉别人，但是你要为这个整体作出贡献，你不能损害这个整体的利益，别人就不再相信你，你就不是这个共同体的一份子，你的存在就没有价值。

不活在别人的评价下，就要有被讨厌的勇气。不怕别人讨厌你，因为他讨厌你对你的生活没有任何影响，完全没有影响，真正的自由就是被别人讨厌。

我为家庭做出贡献就是为共同体作出贡献了，我就已经实现目标了，在任何时候我都得到人生最终目标了而得到满足，于是就得到了幸福，为共同体作出的每一点都是幸福，幸福不是说人生最后临死的时候达到人生目标了，我赚了最多的钱，我觉得我幸福，不是这样，幸福在每时每刻发生的，你做出一个对共同体有意义的事情，这一刻你就获得了幸福，所以幸福对于所有人都是很容易的到的东西。
