# 016To do list

1. **健康是基础。** 身体健康（规律运动、充足睡眠、健康体态、避免久坐、健康饮食），精神健康（稳定情绪、抗压能力、内心平静）。**没有健康，再喜欢的人你无力陪伴，再好的事业你扛不住压力，再多财富也无福消受。**
2. **联结是幸福。** 善待爱人、家人、朋友，善待每一段相遇，少怨憎、多包容，化解爱别离、怨憎会的苦。
3. **成长是生命力。** 成为自己，成长不只是赚钱。而是让自己一年比一年更强、更自由、更有选择权。学习新的能力、提升认知、创造价值、完成自己想做的事情，成长带来的不是满足，而是一种生命力。

**照顾好身体，珍惜重要的人，坚持做有长期价值的事。很多年后回头看：财富是结果、名声是结果、职位是结果，而健康、关系和成长，才是贯穿整个人生的东西。**
