# 027责任

**责任，** 简单来说就是：**一个人对自己、对他人、对社会应该承担的义务，以及对自己行为后果负责的态度。** 责任不是别人强加给你的东西，而是一个人成熟后愿意承担的事情。

对自己负责。这是最根本的责任，维护身体健康、学习成长、管理情绪、为自己的选择承担后果。考试没考好，先反思自己哪里没做好，而不是一味怪题目难。工作能力不足，就主动学习提升。这叫**为自己的人生负责。**

对家庭负责。家庭中的每个人都有自己的责任，父母抚养和教育子女、子女孝敬父母、夫妻相互扶持。责任不一定是轰轰烈烈的大事，很多时候就是说到做到、信守承诺、在别人需要时站出来。

对工作负责。工作中的责任体现为完成自己的任务、保证工作质量、不推卸问题、出现错误及时改正。程序员发现线上 Bug，推给别人 → 缺乏责任感、主动定位和修复 → 有责任感。

对社会负责。人生活在社会中，也承担着公共责任。遵守法律、诚信守约、爱护公共环境、尊重他人。

责任与自由。很多人认为责任会限制自由，实际上**责任越大，自由往往越大。** 因为真正的自由不是想干什么就干什么，而是**有能力承担自己选择带来的后果。** 小孩可以随便玩，但很多事情做不了，成年人有更多选择，同时也要承担更多责任。

责任与担当。责任是客观存在的义务，担当是主动承担责任的勇气。项目出了问题需要解决。主动站出来处理问题。所以常说**责任是身份赋予的，担当是品格体现的。**

人生中的责任。从成长角度看，一个人往往会经历**照顾自己 → 照顾家庭 → 服务社会。** 承担责任的能力越强，能够影响和帮助的人就越多。因此可以把责任理解为**对自己的人生负责，对身边的人负责，对自己的选择负责。** 责任不是负担，而是一个人从依赖走向成熟的重要标志。
