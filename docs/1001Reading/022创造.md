# 022创造

**创造，** 就是**把原来没有的东西变成有，把旧的东西变成新的东西。** 创造不仅仅是发明科技，也不仅仅是艺术创作。凡是通过思考、行动，产生新的价值，都属于创造。

**创造的本质。创造包含三个要素：发现问题、产生新想法、形成新价值。** 发明手机是创造，写一本书是创造，设计一个网站是创造，开发一个 AI 应用是创造，提出一种新的解决方案也是创造。创造不一定是从零开始，很多创造其实是**旧元素的新组合。**

创造与模仿。很多人觉得创造就是完全原创。实际上创造往往从模仿开始。程序员先模仿优秀项目，画家先临摹大师作品，作家先学习经典作品。然后逐渐形成自己的风格。过程往往是**模仿、理解、改进、创造。** 创造与劳动。劳动解决的是把事情做出来。创造解决的是把新的东西做出来。按照需求写代码，是劳动；设计新的产品架构是创造；两者并不矛盾，很多创造都建立在长期劳动基础上。

创造与价值。创造的意义不在于“新”，而在于“有价值”。随便画一条线是新的，但如果这条线解决了问题、表达了思想、带来了美感，它才具有创造价值。所以创造 = 新颖性 + 价值性。人为什么要创造。生存解决活下去，责任解决承担义务，创造解决让世界因为自己的存在而有所不同。一个人通过创造改变环境、改变生活、改变自己、影响他人，从这个角度看，人类文明的发展，本质上就是一代代人的创造积累。生存、责任、创造。这三个词放在一起，可以看成一个成长过程：生存先让自己活下去；责任对自己和他人负责；创造在承担责任的基础上创造新的价值；所以可以概括为**生存是基础，责任是成长，创造是超越。** 一个人如果只关注生存，可能只是活着；能够承担责任，开始变得成熟；能够持续创造价值，则会让自己的人生和周围世界都变得更加丰富。
