# 018《走出孤独》-阿德勒

真正孤独是指无法融入社会，与其他人建立合作关系人，这种人虽然身处人群中，也仍然感觉到和团队、和社会格格不入。

阿德勒看来孤独是由自卑引起的，阿德勒强调自卑感人人都有，它是一种十分正常的情绪。自卑是一个人对自身状态的不满，正因不满，所以我们想要弥补自身欠缺的部分，追求优越感，来超越自我，达到理想的状态，比如一个牙牙学语的孩子，在看到身边大人畅所欲言后，他会感觉自己的无力，因此有了自卑感，这种自卑感就趋势它最求优越感，去学习语言，从而能和周围的人自由沟通。更进一步说，一切文明都是在自卑感的基础上发展起来的，正是因为人类感觉到了自己的无知，又对未来充满期待，所以才会努力去探索自然和宇宙，科学才会得以诞生和发展，人类踏出的每一步行动，其实都是为了克服自卑感和追求优越感，这二者相辅相成，是人类超越自我的原动力。

轻微的自卑感能让我们不断奋斗追求卓越，但是过于强烈的自卑感反而会让我们跑偏方向，产生自卑情结和优越情结。正是这两种情结让我们无法与人简历合作关系，因而我们就会感觉孤独，自卑情结和优越情结是阿德勒心理学的重要概念。

- 自卑情结是一种不正常的心理状态，他是指当一个人面对问题时，不想相信自己有解决的能力，所以产生了自暴自弃的退缩反应。口吃患者苏珊认为自己因为口吃而无法找到好工作，于是她就自暴自弃，不去找工作，就是一种典型点的自卑情结思维，但是拥有健全自卑感的人就会尽力去弥补自己的短板，比如通过刻苦练习说话来治好自己的口吃，并努力积攒自己工作所需技能，为找到好工作做准备，相反她并没有这种改变的勇气，她认为出去找工作，别人肯定会嘲笑她的口吃，所以他就干脆不出去找工作，只待在自己狭小的世界里，最终她就会变得越来越害怕与外界接触，因而感到越来越孤独。
- 优越情结其实也是一种不正常的心理状态，它是指一个人表现自己好像很强大，从而沉浸在愈加的优越感中，以此来掩盖内心的无力和自卑。比如口吃的苏珊因为口吃自卑，但她也无法接受这样的无能的自己，她就暗示自己"我，要不是有口吃，我肯定能够找到一份很好的工作"，据此她在内心幻想出一个没有口吃的职场女强人，并从中获得了优越感，但是显示中的苏珊还是没有改变现状，她依然不敢去融入社会去找工作，她所获的的优越感是虚假的，他只是在自欺欺人而已。再举几个例子，在我们生活中，有这样一种人，他们小时候可能比较穷，自己总是觉得低人一等。于是为了弥补自己的自卑感，他们就会拼命去赚钱，无论是什么手段，只要能够赚到钱就可以，而且他们有钱了，最喜欢的事情就是炫耀，比如他们会追求各种各样的奢侈品，身上总是穿着各种各样的名牌服饰，他们巴不得全世界的人都知道他们是有钱人，以此来显示自己是一个特别的存在，一个总想着要突出自己特别身份的人，必然无法和他人建立平等合作关系，因此也无法友好的融入社会，这样的人会内心孤独，其实也不足为怪了。从上面的例子可看出，虽然自卑情结和优越情节看起来是相反的两个词语，但是二者联系密切。

一个人会因自卑感走向超越自我的道路还是孤独的道路，取决于这个人是否有充足的社会感，社会感强的人，能积极融入社会，去和其他人建立合作关系，缺乏社会感的人，则自顾自己，无法很好的融入社会，与其他人产生联结。走出孤独的方法，首先就在于自己需要改变的是目标，其次要重新设定目标要能够体现自己充足的社会感，当你学会和其他人良好联结，并在与其他人的关系中获得成长时，即使你独来独往，也一样能活出自己的价值，重新给自己设定一个有意义的目标去和其他人合作。

— 2026.6.1 深圳
