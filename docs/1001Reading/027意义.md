# 027意义

**意义，** 就是**一件事、一个人、一种经历为什么值得存在，以及它对自己或他人产生了什么价值。** 简单说**意义回答的是“为什么”。**

意义与价值，两者很接近，但不完全相同。**价值是** 有什么作用，**意义是** 为什么重要。一份工作价值是赚钱养活自己，意义是实现理想、帮助家人、服务社会；一本书的价值传递知识，意义是改变一个人的认知和人生。

人为什么会追问意义。当生存问题解决后，人往往会开始思考：我为什么活着？我为什么工作？我为什么学习？我追求这些到底为了什么？这种追问，本质上是在寻找意义。所以：**意义是人类区别于许多动物的重要特征之一。** 动物更多关注生存。人除了生存，还会追问生命的方向。

意义的来源。意义通常来自几个方面。

- 爱与关系，家人、朋友、爱人，很多人努力奋斗，不只是为了自己，而是为了自己在乎的人。
- 成长，学习新知识、提升能力、克服困难、成长本身会让人感受到意义。
- 创造，写一本书、开发一个产品、创办一家公司，创造意味着世界因为你的存在而发生了一点改变。
- 贡献，帮助别人、服务社会、传承经验、很多人在帮助他人的过程中找到意义。

意义不是别人给的，很多人会问人生的意义到底是什么？事实上不同的人答案可能不同。

- 有人认为家庭最有意义
- 有人认为探索真理最有意义
- 有人认为创造事业最有意义，

因此**意义往往不是被发现的，而是在生活中逐渐建立的。**

意义与幸福，幸福是一种感受，意义是一种方向。

- 打游戏可能很快乐；
- 学习一门困难技术可能很辛苦，但几年后回头看成长带来的意义往往更持久。

所以心理学中经常认为：**快乐让人享受当下，意义让人坚持长期。**

**人生最大的意义，不一定是获得什么，而是在不断成长、创造、爱与被爱中，让自己的生命变得更加丰富而充实。**
