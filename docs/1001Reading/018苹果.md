# 018苹果

## 包豪斯与禅宗对苹果设计的影响

包豪斯最著名的一句话：Form follows function，形式追随功能。意思是不要为了装饰而装饰。设计首先服务于功能。例如去掉多余按钮、去掉复杂装饰、强调几何结构、强调实用主义

年轻时的乔布斯：学习禅修、阅读东方哲学、去印度旅行，他后来多次提到东方哲学对自己影响巨大。他认为：专注来自于说“不”。很多人以为创新是不断增加，乔布斯认为创新是不断删减。**结合效果** 苹果产品哲学 = **极简而强大，** 禅宗保证界面简洁、用户感受轻松，核心理念**复杂留给系统，简单留给用户**

## 苹果的文化：**极致追求卓越，用简单创造伟大。**

但如果深入看，苹果文化其实由几个核心信念构成。

**用户体验高于一切。** 很多公司文化是技术第一、销售第一、利润第一，苹果是用户体验第一。在苹果内部经常会出现这样的决策成本更高、开发更难、利润更低，但只要体验更好，仍然会做。例如自研芯片、Retina屏幕、MagSafe、多设备协同，因为苹果认为用户不应该为技术妥协。

 **对卓越近乎偏执。** Steve Jobs 曾说：Real artists ship，真正的艺术家会交付作品。但同时他又极度追求细节。在苹果一个圆角、一个动画、一个包装盒，都可能被反复修改几十次。很多离职员工形容苹果像一家工程公司，也像一家设计公司，更像一家艺术公司。

**专注（Focus）。** 乔布斯回归苹果后砍掉了大量产品线。因为他认为：Focus is saying no，专注意味着拒绝。苹果文化有一个特点敢于放弃，不做低端机海、大量产品线、各种试验品，只做少数重要产品。

**简单至上。** 苹果相信简单比复杂更难。因为简单的背后是更深的思考、更多的删减、更高的设计能力，所以苹果文化里有一个习惯：不断问：这个按钮能删吗？这个菜单能隐藏吗？这个流程能缩短吗？

**跨学科合作。** 苹果和很多科技公司不同。它不相信工程师做工程、设计师做设计、市场做市场，而是：工程、设计、产品、运营、营销，一起参与产品定义。因此苹果更容易做出完整体验。

**保密文化。** 苹果是全球最重视保密的公司之一。很多团队不知道其他团队在做什么，项目代号保密，新产品长期封闭开发，目的是避免噪音，让团队专注创造。

**改变世界的使命感。** 苹果文化最特别的一点。很多公司目标赚钱、增长、市场份额，苹果长期强调：创造伟大的产品。1997年的经典广告：Think Different，这句话至今仍是苹果文化的精神内核。苹果希望员工相信自己不是在制造电子产品，而是在改变人们工作、创造和生活的方式。

所以苹果文化最核心的三个关键词其实是：专注（Focus）、简单（Simplicity）、卓越（Excellence）这三个词组合起来，就是苹果几十年来最稳定、最强大的文化基因。**很多公司学习苹果，会学它的设计、营销、发布会。但真正难学的是这种文化：为了做到简单，愿意付出复杂；为了做到卓越，愿意付出成本；为了保持专注，愿意放弃机会。** 这才是苹果持续创造优秀产品的根源。
