# 005家庭教育

家庭教育拆成两个部分：**父母教育自己（父母教育）、父母教育子女（子女教育）** 很多家庭的问题，不是第二部分做得不够，而是第一部分根本没做。

### 一、先教育父母自己

很多人认为：生孩子 → 教孩子，但实际上应该是：学做父母 → 生孩子 → 教孩子，可惜现实中很多人跳过了第一步。

**学会自省。** 每天问自己今天有没有乱发脾气？今天有没有认真听孩子说话？今天有没有把自己的情绪发泄到家人身上？很多父母总在研究孩子哪里有问题？却很少研究我哪里有问题？家庭成长往往从这里开始。

**管理自己的情绪。** 孩子最怕的不是父母严格，而是一会儿温柔，一会儿暴怒，完全不可预测。孩子会长期处于紧张状态，所以父母要学习觉察情绪、延迟发火、理性沟通。

**保持终身学习。** 时代变化很快。父母不能总是我当年就是这样过来的。例如AI时代、互联网时代、短视频时代，孩子面临的问题和父母年轻时完全不同。

**经营夫妻关系。** 很多人以为家庭教育 = 教孩子，其实家庭教育首先是夫妻关系。孩子天天看见，爸爸怎么对妈妈，妈妈怎么对爸爸，这就是他的婚恋启蒙课。孩子主要靠观察学习。

### 二、教育孩子

当父母把自己建设得差不多了，再谈教育孩子。

**第一阶段：0~6岁。** 重点：安全感，让孩子知道：我犯错了，父母依然爱我。不是：考100分才爱你。习惯培养：早睡早起、阅读习惯、整理习惯、运动习惯比学多少知识更重要。

**第二阶段：6~12岁。** 重点：责任感，例如自己整理书包，自己整理房间，不要全部代劳。规则意识让孩子明白：自由来自责任。而不是想干什么就干什么。

**第三阶段：12~18岁。** 重点：独立思考不要只问：听懂了吗？更要问：你怎么看？

**价值观。** 帮助孩子思考：什么是诚信、什么是责任、什么是尊重，这些东西比分数影响更长远。

### 三、家庭教育最核心的原则

**不是控制，而是示范。** 孩子不会成为你要求的样子，更容易成为你实际表现出来的样子。例如父母天天玩手机，却要求孩子读书。效果通常很差。但如果父母每天看书、学习、锻炼、守信用，孩子会潜移默化受到影响。

### 四、一个更完整的家庭教育观

很多人把家庭教育理解成父母站在上面教育孩子。我更赞同另一种理解：全家共同成长，父母子女互相影响互相学习。父母教孩子生活经验；孩子也会让父母接触新知识、新观念。一个好的家庭，不是父母永远正确、孩子永远服从，而是每个人都保持学习和成长。如果用一句话总结：家庭教育最重要的不是把孩子培养成什么样的人，而是让整个家庭形成一种“持续学习、持续反思、互相尊重、共同成长”的氛围。这样的家庭，父母和孩子都会越来越成熟。
