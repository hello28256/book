# 004不反应练习

八苦（无能为力）：

- 生、老、病、死、
- 爱别离、怨憎会，求不得、
- 五蕴炽盛（色：形体、受：感受、想：思想、行：行为、识：意识）根本

不要做多余没有意义的反应：对一些无能为力的事情，不要做无谓的反应（例如：排队的时候抱怨）

底层欲望：生存欲、睡眠欲望、食欲、性欲、惰欲、享乐欲、认可欲

要冷静客观的看待自己的烦恼，才能解决自己的烦恼，你不能解决掉一个你不是很了解的东西，太多人不了解自己的烦恼，所以一直在烦恼，当你真正理解自己的烦恼的时候，你可能会发现你的烦恼很有可能是一个无聊的想法

解决烦恼：

1. 把这个烦恼的问题说清楚，真正理解烦恼
2. 闭上眼睛体会自己的感受
3. 将烦恼分类
   1. 欲望型烦恼：烦恼的根源是一个不合理或者过分的要求和期待
      1. 只能放下了，不要有那么多欲望、期待、要求
   2. 愤怒型烦恼：遇到什么事情产生很生气的情绪，如被否定、失败、挫折、压力
      1. 冷静下来、愤怒是不能解决任何问题的
   3. 妄想型烦恼：对未来悲观的妄想（想太多，会不会发生不知道，但是已经开始折磨自己了）
      1. 不要瞎想，想象终究是想象，不是客观事实
   4. 自卑:（经历了太多的失败或者糟糕的家庭环境和学校教育，自卑的人由于经历过太多的打击和太大的打击，所以他们就比一般人更加渴望成功，渴望被别人认可，如果这种渴望无法的到满足的话，他们会变得异常有攻击性，他们会认为别人的成功一定使了什么不公平的手段，不然为什么我们做一样的事情，为什么我不成功呢，这是自卑人的一种表现，这也就是他的问题，他不认为自己有问题，而是认为所有的问题都在环境当中，在别人身上，所以他会去攻击周围的一切，甚至攻击自己，自卑的人还有一种类型就是逃避型，现实逃避，逃避型往往会把注意力转移到一些非常容易实现的事情上赌博依存、药物依存、酒精依存、食物依存、购物依存，无论到什么时候都不要自卑，这是一种非常危险的状态）
   5. 如何防止自卑、烦恼发生呢？
      1. 出去走走，分散注意力，人特别怕在一个没有变化的环境中待时间太长，因为你的环境不发生变化的话，你的注意力就会集中到你的想象上面去，就会产生各种妄想。（10分钟）
      2. 看看这广阔的世界，人看到广阔的场景，就会觉得自己是渺小，这就是为什么很多高僧喜欢在大山修炼冥想，环境越宽阔就会觉得自己越渺小，你的那些事情就不是事，多思考些格局大的问题，原先关注的鸡毛算皮的事情，原来的事情就不是事情，不再是烦恼了
      3. 要学会接受自己，人的很多烦恼源自不接受现实的自己，你觉得你不应该是这样的，当你接受你自己，我就是这样的，那就不再有烦恼了 ，接受不是放弃，而是能正视自己的问题，而且你接受自己，才能认可现在你的成绩，这个世界不是只有胜和败之分，我们是有很多路可以走的

积极正面的事情：

1. 慈：对周围的众人抱有慈爱仁慈之心，希望他们得到幸福
2. 悲：对于众生的痛苦要有同情怜悯之心
3. 喜：对他人的成功和幸福感到欣喜，而不是嫉妒和竞争
4. 舍：放下执着、放下欲望、不要愤怒和妄想
