# 011吸引力法则

吸引力法则提供了一个改善你自己生活的思想方式，这个思想根源就是，你希望你的生活变成什么样子，你的生活就有可能慢慢变成那个样子，不论好坏。也就是你生活状态是，你思想的体现，而且是有一定根据的，你的思想和想法，和你现在的状态和未来的状态是有直接关系的。吸引力法则有两个法则：物以类聚、思想现实化。

## 物以类聚

根据吸引力法则，人会严重受到周围人的影响，而渐渐变成和周围人一样的人，这个一样是指你的性格，你的思考方式，你的逻辑，你的人生观，世界观，都会渐渐与你接触到的人变得越来越像，这个就是吸引力法则的物以类聚。物以类聚的结果就是你会受到周围人的影响，渐渐过上和周围人一样的人生，有很多证据可以证明你和周围的人越来越像。我们应该对于别人的幸福，别人的成功，应该是祝福的态度，因为不祝福，你就到对立面去了，你就很容易不幸，就是你不要和一个幸运的人成为敌人，如果你能和这些成功的人士接触，你的思维方式就会渐渐接近他们，当遇到问题的时候，你解决问题的方式就会和他们类似，就会得到类似的结果，人生就会发生类似的改变。所以为什么要去图书馆学习，去健身房锻炼，这也就是为什么在一个国家学不会外语的原因，环境没有嘛，周围的人都不说外语，所以你接触的人直接决定你的人生，这就是吸引力法则中的物以类聚的法则。相似的东西就会互相吸引，你想成为什么样的人，你就接触什么人就好了。

## 思想现实化

思想现实化就是说我们的想法具有实际吸引力，由于这个吸引力现实世界会不断靠近你的想法，就是你想的事情会靠近你。所以你要是想成为成功人士、成为有钱人士的话，你就要一直想着这个事，直到自己真的信了为止，如果你有半点怀疑的话，你骨子里并没有相信自己会成功，不相信自己会成功还好，大不了不成功。如果你不相信自己会成功的话，它会吸引失败的结果，它会反向吸引。所以吸引力法则中最重要的就是"你究竟信什么"，表面功夫是没有用的，假装自己是有钱人是没有用的，要深信不疑。吸引力法则为什么不叫成功法则，是因为它吸引好的也吸引不好的东西，终究是吸引，取决于你的真实想法。比如说愤怒、压力这些负面情绪，其实是从一点一点的负面情绪开始，不断的吸引更多的负面情绪，最总导致你无比的愤怒，无比的压力，最终抑郁生病，爆发一次没什么用，根本上是你的想法的问题，你的想法不转变是不行的。所以生病通常是你想法有问题的一种标志，不会有人每天都很愤怒很抑郁，但很健康，所以你生病了，就是在提醒你最近的想法有问题，正所谓病由心生。

解决的唯一办法就是把负面能量的想法扼杀在摇篮之中，就是任何和好的想法和不好的想法都是一点点开始，在这个不好的想法出现的一瞬间，就把它扼杀掉，转移注意力，不去看他了，然后植入一个正面的想法，这个正面的想法就会不断吸引更多的正面的想法，正能量的东西，你的生活就会越来越好，身体就会越来越健康，生了病的人都会痊愈。所以积极健康的想法足以治百病，其实有一个很好的证明就是安慰剂效益，给病人吃一个糖豆或者面团，这种伪药总会有一部分病人出现并且好转的迹象，还有一个反安慰剂效益，就是你认为这个药没用，那它是神丹妙药都没用，治病是靠药，能不能治好是看心情，由好的心情，积极正面的想法，可以带来好的身体，好的人生，所以要保持好的心态。佛教中为什么要强调心无杂念，是因为杂念会吸引更多的杂念，烦恼会吸引更多的烦恼，痛苦吸引来更多的痛苦，这样的话你的人生只会越来越糟；相反你向往美好、向往幸福的话，你这种正面的思想，就会吸引更多的思想，更多的幸福。我们的大脑是一个脑电波发送装置，而这个世界是脑电波接受装置，你想的会被映射在你眼前，所以你想要什么世界，完全由你的大脑决定。由于吸引力法则，也就是你的想法可以改变你的人生的，所以你现在的人生，其实是由你过去的想法创造的，而你未来的人生将由现在所创造，所以从现在开始有一个积极正面的想法的话，未来就不会糟。

## 如何实施吸引力法则：主要分为三步

1. 第一步就是给自己设定一个目标，假设自己想要的一个结果，这个目标要明确，不能模糊，这个目标不能是愿望，"我想成为有钱人"这样的目标是不可以的，要明确要准确，愿望就说明你心里是不相信它能实现，无效还好，吸引力法则，它就会把那个更糟糕的结果带来，所以如果你没有打心底相信，你会成为有钱人，你会直接成为穷人。大部分人无法实现吸引力法则，设定的目标的是忽视所有副词的，比如:"我想成为一个非常非常有钱的人"，这个非常非常是无效的。
2. 第二步就是相信自己已经有了，已经实现了，只是这个目标还不在你手里，坚信自己已经有了，吸引力法则中最困难的一步就是你要对自己这个目标深信不疑很多人，相信自己有了，不然就不会起作用，其实很多人在无意识形态下让这个吸引力法则失效了，就是他们都很自卑，自卑这个东西是处于潜意识的，不是意识，你可以在意识层面面前装作很自信，但是自卑是在骨子里的，而吸引力法则看的就是你骨子里的东西，表面上说自己会成功，但是内心是自卑的话，那同样是不会成功的。那么怎么才能解除自卑呢，自卑往往来自于经历，你可能失败过很多次你可能就会自卑，你可能生活条件不好你就会自卑，你只要忘记这些经历就好了，一切从零开始，一切从零开始就不再有自卑，你要想成功必须忘记这些东西。只有消除自卑之后呢，你才会真正开始相信自己已经成功了，有一个可以检验是否真的信，就是真的信你就会发自内心感觉舒服，但如果不是真的信的话，你就会有各种违和。我真的有钱了，我真的有钱了，又觉得自己生活没有发生变化啊，真正信的人已经开始享受了，躺地上已经开心了，就好像已经做梦数钱了 ，这叫真的信。
3. 保持好状态，等着接住。

## 一些技巧

1. 要训练自己瞬间转移注意力的能力，所有不好的想法，负面的情绪都是从一点点开始的，我们怎么去扼杀，去不想他了实际还是在想，最好的方法就是转移注意力，你把注意力转移到其他地方都可以，你想到美好的东西，生活就会越来越美好越来越幸福。
2. 不要想太远，如果你的目标太远，就要花越长的时间，结果中途开始怀疑自己。
3. 要把你的目标视觉化，你想到你有钱是怎么的状态，穿什么样的衣服，住在什么样的地方，开什么样的车，和什么样的人接触，在哪个城市，想得越清楚，这个目标实现的速度就越快实现，太模糊的话，这个世界都不知道你要找什么样的目标，很难吸引来。
4. 行动，它过来很慢，但是你可以过去，不要光想不做，不要优柔寡断，有些许犹豫就是不自信，就是不相信它会实现，相反失败的结果就会被拽来，这个地方可以检验你是否相信你自己能够成功，就是你真的信了，你为你的目标努力的生活，你就不会觉得累，就不会觉得痛苦。
5. 感谢，是一个非常正面的东西，它能吸引来各种各样正面的东西，你感谢别人，别人就会被你吸引来，你感谢钱，钱就会被你吸引来，你想要什么就感谢什么就好了，我们要一直心怀感激之情，不用谢天谢地，谢谢你周围的一切就好。
