# 029挫折

**挫折，** 就是**人在追求目标的过程中，遇到阻碍、失败、打击或不如意的经历。想要的得不到，计划好的做不到，努力后没达到预期，** 这就是挫折。

**挫折是人生的常态。** 几乎每个人都会遇到考试失利、求职被拒、创业失败、恋爱受挫、人际关系破裂、身体疾病。你投了很多简历，却一直没有面试结果，这就是一种挫折。它意味着现实暂时没有按照你的期待发展。

**挫折的本质。** 挫折并不等于失败，它本质上是**理想与现实之间的落差。** 想一个月学会某项技术，结果三个月还没掌握；希望对方主动联系自己，结果一直等不到消息。这种落差会带来：失望、焦虑、痛苦、愤怒。

**挫折的价值。** 虽然难受，但挫折往往能带来成长。它会让人认识现实知道世界不会完全按照自己的想法运行；发现不足看到能力、方法或认知上的缺陷；增强韧性，学会面对困难而不轻易放弃。很多能力不是顺境培养出来的，而是在挫折中磨炼出来的。

**面对挫折的三种态度：**

第一种：逃避。抱怨、自暴自弃、责怪别人、问题通常不会因此消失。
第二种：硬扛。不承认痛苦、强行压抑情绪，短期有效，但容易积累压力。
第三种：接纳并调整，承认这件事没有达到预期，然后思考为什么会这样？哪些因素能改变？下一步怎么做？这是更成熟的应对方式。

**挫折与成长，** 一个很形象的比喻：**读书、思考、成长，不是让挫折消失，而是让自己从一个杯子变成一个盆、一个桶，甚至一片海。** 同样一块石头丢进杯子里，水会溅出来；丢进桶里，波动变小；丢进大海里，几乎掀不起什么浪花。挫折没有变，变化的是承载它的自己。
