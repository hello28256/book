# 025自由

**自由，** 就是**能够按照自己的意志进行选择和行动。** 简单说**自己决定自己的人生，而不是完全被别人或环境支配。**

---

**自由不等于想干什么就干什么，** 很多人认为自由就是没有任何限制。实际上并不是，你有开车的自由，但要遵守交通规则；你有说话的自由，但要承担说话的后果；你有选择工作的自由，但要接受选择带来的结果。**自由永远伴随着责任。**

**自由的层次**

- **身体的自由，** 能够自由行动。不被囚禁，不被强制控制，这是最基础的自由。
- **时间自由，** 不用把全部时间卖给生存，日程自己说了算，不用被动加班、被动应付琐事。
- **经济的自由，** 拥有一定的物质保障，有收入来源，能满足基本生活需求，经济压力越大，选择空间往往越小。
- **思想的自由，** 能够独立思考。不盲从权威、舆论、群体压力，有自己的判断能力。
- **精神的自由，** 这是更高层次的自由。不被贪婪控制、不被恐惧控制、不被虚荣控制、不被执念控制，外部环境可能有限制，但内心依然保持自主。

**自由与欲望，** 很多人觉得满足所有欲望就是自由，实际上一个人无法控制欲望，那么他反而不自由。想玩游戏就停不下来、想刷短视频就停不下来、情绪来了就失去理智，看似随心所欲，实际上是**被欲望牵着走。** 真正的自由是**既可以满足欲望，也可以拒绝欲望，** 拥有选择权，而不是被迫行动。

**自由与责任，** 责任和自由看似对立，其实相互依存。小孩责任少，但很多事情不能决定；成年人责任更多，但拥有更多选择权。**责任越大，往往意味着自由越大。** 因为你有能力承担选择带来的后果。

**自由与成长，** 人的成长过程，也可以看成获得自由的过程，婴儿依赖父母、学生依赖家庭和学校、成年后逐渐独立。自由不是别人给予的，很多时候是靠能力、认知、金钱、自律一步步争取来的。**拥有能力面对现实，拥有智慧管理欲望，拥有勇气承担责任，然后按照自己的价值观去生活。**
